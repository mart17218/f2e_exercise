If want to open eslint for .vue files,
1. npm install eslint-plugin-html
2. change webpack.base.conf.js
```
test: /\.js$/,  ->  test: /\.(js|vue)$/
```
3. Add config to .eslintrc.js
```
plugins: [
   "html"
 ],
 ```
